Strokes Gained is a golf statistic used to determine the quality of a given shot. It compares the expected number of strokes from the ball's location pre-shot to the expected number of strokes post-shot.

For a more detailed description of strokes gained, consult pgatour.com

Use the following tool to calculate the strokes gained for any golf shot.

NOTE: For putts, enter the distance in feet as a whole number. If the shot went in the hole, put 0 as the remaining distance and 'hole' as the finishing lie
